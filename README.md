Overview-This full-stack application allows authenticated users to create, view, and edit public tasks.
All users can see tasks posted by any user, while only Admins are allowed to delete tasks.

##Tech Stack

#Frontend
React (Vite)
Axios
Modern UI Styling

#Backend
Node.js + Express
JWT Authentication
MongoDB + Mongoose
CORS Protected

##Installation & Setup
git clone https://github.com/Mahekshaikh389/BackendProject
cd BackendProject
cd backend
npm install
npm start

##Install Frontend Dependencies
cd ../frontend
npm install
npm run dev
